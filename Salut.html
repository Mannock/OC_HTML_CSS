Les images

Les formats

Il existe principalement 3 formats d'images que l'on va retrouver sur le web:

  • Les formats JPEG: pour toutes les photos
  • Les formats PNG: pour toutes les autres images qui ne sont pas des photos
  • Les formats GIF: pour les images animées

Pour insérer une image, on utilise la balise img/. On lui ajoute les deux attributs obligatoires:

  • src: le chemin de l'image. Il peut etre asolu (http://site.com/fleur.png) ou plus souvent relatif ("src ="images/fleur.png)
  • alt: un court texte de description, pour l'accessibilité et la sémantique.

Les images doivent toujours se trouver à l'intérieur d'un paragraphe.
